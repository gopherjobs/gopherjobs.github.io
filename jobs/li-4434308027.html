Backend Engineer (Go/JS) – DEX & Web Services

About The Role We are looking for two Backend Engineers to build the scalable infrastructure for a decentralized exchange and suite of web services integrated with the Bitcoin ecosystem. This role focuses on the Service Layer — the high-concurrency engine that handles API requests, manages off-chain order matching, and interfaces with blockchain nodes. Core working hours: 10am–1pm CET.

Responsibilities

  • Service Development: develop and maintain the backend microservices that power the exchange and the wider web service ecosystem
  • Order Management: implement the logic for off-chain order books, trade matching, and real-time liquidity monitoring
  • Node Integration: build the middleware that connects Go/JS services to the core blockchain network
  • Scalability: optimize backend services for low latency and high availability
  • Security: implement industry-standard security practices for handling sensitive data

Requirements

  • Strong proficiency in Go (Golang) or Node.js (JavaScript/TypeScript) with deep understanding of asynchronous programming and microservices
  • Experience building DEX components, API gateways, or indexing services
  • Expertise in designing and implementing RESTful or GraphQL APIs
  • Experience with both SQL (PostgreSQL) and NoSQL (Redis/MongoDB) for real-time data tracking
  • Ability to build thread-safe systems handling thousands of concurrent user requests

Benefits

  • Health & Prevention: Comprehensive, high-tier medical insurance with a focus on preventative medicine, annual executive health screenings, and diagnostic check-ups.
  • Fitness & Wellness Programs: Monthly fitness allowances for premier gyms and studios, plus internal wellness initiatives including yoga, ergonomics workshops, and stress management.
  • Competitive Compensation: Salary paid in your choice of fiat, Bitcoin, or stablecoins. ML token allocation and vesting for senior roles.
  • Relocation Support: Full assistance with UAE residency visas and relocation to Abu Dhabi for on-site roles. We handle the paperwork.
  • Flexible Working: Structured flexibility across on-site, hybrid, and fully remote roles. Core overlap hours respected across time zones.
  • Global Team Meetups: Periodic gatherings with the full team around the world. The remote team is never remote from the culture.
