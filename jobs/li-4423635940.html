Description

Automating software update of Large Engines

As a Software Engineer at Everllence, you will play a key role in the digitalization of the world’s leading ship engine brand, helping to reduce greenhouse gas emissions.

This is an opportunity to apply your skills to one of the most important challenges of our time—addressing climate change—and to become part of the movement towards a carbon‑free future.

You will play an important role in:

  • Design and development an end-to-end “Over-The Air” Update Solution for our Engine system
  • Develop and test your own software, including writing unit test, integration test, and system test
  • Building virtual test setup using Virtual Box running in Linux/Windows environment for Integration test and automate processes.
  • Contribute to building physical test environments with real HW (simulating Engine room setup)
  • Implement cyber security enhancements to existing software modules
  • Create CI/CD pipelines using GitHub Action and playing DevOps role sometimes

We imagine that you:

  • MSc or BSc in Software Engineering, Computer Science, or a related technical field
  • Experience with modern software development using C#, PowerShell, Python
  • Experience with virtualization technologies (e.g., VirtualBox or similar)
  • Experience with CI/CD and building pipelines (e.g. GitHub Action)
  • Experience working in Windows/Linux environments, including client/server application development
  • Familiarity with test frameworks and automated testing infrastructure.
  • Strong collaboration and communication skills, with the ability to work in cross-functional engineering teams.
  • Fluent in written and spoken English, with the ability to work independently

Welcome to Edge Platform Team You will be a part of the Edge Platform team within the Automation department in Copenhagen. The Edge Platform team holds end-to-end responsibility for the design and development of Over-the-Air update functionality for engine systems. This includes developing custom Windows images for PCs running Engine Control System, orchestration modules on edge, and cloud-based solutions for transferring software artifacts to vessels.

The Edge Platform team is a diverse group of software engineers with expertise across cloud development, PC software development (Windows & Linux), user interface design, cybersecurity, data management etc.

Your new team will consist of 12 engaged Software Engineers based in Copenhagen who, together with a small group of consultants, are responsible for building a scalable orchestration platform that enables “Over-the-Air” updates across an entire fleet of vessels.
