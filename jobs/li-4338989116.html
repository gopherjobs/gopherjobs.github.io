Responsibilities DevOps & Cloud Infrastructure

  • Design, build, and maintain cloud infrastructure on Azure/AWS/GCP for multiple projects.
  • Implement and manage IaC (Infrastructure as Code) using Terraform / Bicep / CloudFormation.
  • Build scalable container-based environments using Docker, Kubernetes (AKS/EKS).
  • Optimize performance, cost, and reliability of cloud workloads.

CI/CD Automation

  • Build, maintain, and optimize CI/CD pipelines (Azure DevOps, GitLab CI, GitHub Actions, Jenkins).
  • Implement automated build, test, deploy flows for microservices, web apps, mobile apps.
  • Enforce release standards, environment consistency, and deployment quality.

Security, Compliance & Observability

  • Implement security best practices: IAM, secrets management, network security, vulnerability scanning.
  • Manage monitoring, logging, alerting systems (Prometheus, Grafana, ELK, Loki, CloudWatch, Azure Monitor).
  • Set up backup, disaster recovery, autoscaling, and failover strategies.

Support for Development Teams

  • Work with developers to improve performance, scalability, and deployment processes.
  • Troubleshoot environment issues, production incidents, release problems.
  • Support performance testing, load testing environment setup.

Operational Excellence

  • Participate in incident management, root-cause analysis, long-term remediation.
  • Document system configurations, runbooks, deployment guidelines.
  • Promote DevOps culture and automation mindset inside project teams.

Skills & Qualifications

  • Kubernetes (K8s)
  • DevOps
  • CI/CD
  • Docker

Technical Expertise

  • 3–7 years of DevOps or Cloud Engineering experience.
  • Strong hands-on with at least 1 major cloud (Azure preferred, AWS/GCP nice to have).
  • Solid knowledge of:
  • Docker, Kubernetes (AKS/EKS)
  • Terraform, Bicep, or other IaC tools
  • Linux administration & shell scripting
  • Networking: VPC/VNet, load balancers, DNS, firewalls
  • CI/CD systems: Azure DevOps, GitLab, Jenkins, GitHub Actions
  • Experience managing production systems with high availability.

Programming & Automation

  • Familiarity with scripting languages: Bash, Python, PowerShell.
  • Experience automating workflows, deployments, monitoring tasks.

Soft Skills

  • Good English communication (speaking with foreign clients if needed).
  • Strong problem-solving, troubleshooting, and analytical thinking.
  • Ability to work independently and manage multiple environments.
  • Ownership mindset; reliable and proactive.

Nice-to-have

  • Experience with AI/ML workloads (GPU, model deployment, inference pipelines).
  • Knowledge of RPA/automation workflows.
  • Experience with serverless (Azure Functions, AWS Lambda).
  • Experience with security compliance (SOC2, ISO27001).
  • Experience in fintech, manufacturing, retail, or high-load systems.
