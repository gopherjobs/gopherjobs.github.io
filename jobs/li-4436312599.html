Role Overview:

We are hiring for one of our clients, seeking a Software Engineer, Financial Platform to work on a full-time basis, contributing to the development and maintenance of core financial systems. Key Responsibilities:

• Collaborate with cross-functional teams to integrate third-party financial APIs and payment gateways.

• Implement robust security measures to ensure compliance with financial regulations and data privacy standards.

• Write clean, maintainable, and efficient code following best practices in software engineering.

Required Skills & Qualifications:

• Proficiency in programming languages such as Java, Python, or Go for backend development.

• Experience with financial transaction systems, payment processing, or related domains.

• Strong understanding of distributed systems, microservices architecture, and API design.

• Familiarity with databases like PostgreSQL, MySQL, or MongoDB for financial data storage.

• Knowledge of cloud platforms (AWS, GCP, or Azure) and containerization technologies (Docker, Kubernetes).

• Ability to write unit and integration tests to ensure code reliability and performance.

• Experience with version control systems like Git and CI/CD pipelines.

• Understanding of security best practices and financial compliance requirements.

This role offers a unique opportunity to work with a global leader in the Financial Services industry, contributing to the modernization of financial infrastructure. The position involves building scalable solutions that power real-time financial transactions for millions of users worldwide.
