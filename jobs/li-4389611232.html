Description

We are seeking a Senior Golang Developer to design and maintain backend services, APIs and integrations supporting HashiCorp Vault and related platform components.

In this role, you will collaborate with architects, product managers and engineering teams to deliver reliable, scalable solutions in high-use environments, contributing to code quality and system performance. You will work with containerization and orchestration technologies, AI tools, version control systems and cloud platforms as part of distributed engineering initiatives.

This position is based in Bulgaria, and EPAM provides relocation support for candidates who meet the role requirements and are currently located in Turkey. Learn more about the benefits EPAM Bulgaria provides in the description below.

 

Responsibilities

  • Design and maintain HashiCorp Vault plugins, backend services, or APIs using the Go programming language
  • Collaborate with the stakeholders to align technical solutions with requirements
  • Coordinate with engineers to deliver scalable and high-performing results during high usage
  • Conduct code reviews to enhance code quality, performance, and maintainability
  • Identify and resolve issues reported by stakeholders or internal teams
  • Remain informed about trends in the Go community to implement process improvements

Requirements

  • 5+ years of experience in developing backend services or APIs using the Go programming language
  • Expertise in the Go programming language, including concurrency, channels, and error management
  • Familiarity with containerization tools such as Docker and orchestration tools like Kubernetes and Helm
  • Proficiency in version control systems, particularly GitHub
  • Strong communication and collaboration abilities
  • B2+ English level proficiency

Nice to have

  • Proficiency in using cloud platforms such as Google Cloud Platform or AWS
  • Knowledge of microservices architecture and distributed systems
  • Understanding of agile methodologies such as Scrum or Kanban
