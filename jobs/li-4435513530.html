🔵 About the Role

We are looking for a highly skilled Senior Backend Developer with strong hands-on experience in Python, REST APIs, microservices, FastAPI, Flask, SQLAlchemy, and SQL databases.

This is a full-time, fully remote opportunity for a senior-level backend engineer who has experience shipping reliable, production-quality code and working in a fast-paced startup or product environment.

You will play a key role in designing, building, and maintaining scalable backend services and APIs. We are looking for someone who takes ownership, writes clean and maintainable code, follows strong engineering practices, and is comfortable contributing within a collaborative development workflow.

🔵 Key Responsibilities

  • Design, build, and maintain scalable backend services using Python.
  • Develop and improve reliable REST APIs and microservices for production environments.
  • Work with Python web frameworks such as FastAPI and/or Flask.
  • Build clean, efficient, and maintainable data-access layers using SQLAlchemy and relational SQL databases.
  • Write production-quality code with a strong focus on performance, reliability, maintainability, and scalability.
  • Create and maintain automated tests using pytest to ensure code quality and stability.
  • Containerize services and support consistent development workflows using Docker.
  • Participate actively in code reviews, GitHub pull requests, and collaborative engineering discussions.
  • Follow clean-code, testing, linting, and version-control best practices using tools such as Git, GitHub, flake8, and pytest.
  • Troubleshoot, debug, and resolve backend issues across development and production environments.
  • Work effectively within a remote team while maintaining clear communication, ownership, and delivery quality.

🔵 Required Skills & Experience

  • 5+ years of professional experience building and shipping production-quality backend software.
  • 5+ years of hands-on Python development experience.
  • 3+ years of experience building REST APIs and microservices.
  • 3+ years of experience working with Python web frameworks such as FastAPI and/or Flask.
  • 3+ years of experience with SQLAlchemy and relational SQL databases.
  • Strong understanding of backend architecture, API design, database interaction, error handling, and performance considerations.
  • Strong experience with modern software development practices, including:
  1. Git
  2. GitHub pull requests and code reviews
  3. Docker
  4. pytest
  5. flake8
  6. Clean, maintainable, and testable code practices
  • Previous experience working in at least one startup, scale-up, or fast-paced product environment.
  • Ability to work independently, take ownership of backend components, and contribute effectively as part of a remote engineering team.

🔵 What We Are Looking For

  • A senior backend engineer who has built real-world Python services used in production.
  • Someone who understands the difference between writing code that works locally and delivering backend systems that are reliable, maintainable, tested, and ready to scale.
  • A developer who is comfortable with ownership, code reviews, pull requests, debugging, and continuous improvement.
  • A strong team player who can communicate clearly and work effectively in a remote environment.
